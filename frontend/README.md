# Figma Design Site

This is the new Figma design implementation. It's currently a standalone frontend with no backend integration.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173`

## Note

This is a static prototype with no backend integration yet. Backend capabilities will be added incrementally.

## Structure

- `src/components/` - Main application components
- `src/components/ui/` - Reusable UI components (Radix UI based)
- `src/styles/` - Global styles and CSS
- `src/guidelines/` - Design guidelines and documentation
